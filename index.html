Weather Dashboard Project

For this project, I developed a weather dashboard application that provides users with real-time weather data by fetching information from the OpenWeatherMap API. Users can search for a city and instantly view the current weather conditions, a 5-day forecast, and other relevant details like humidity and wind speed.

The weather dashboard was designed with user interaction in mind, featuring dynamic updates of weather information based on city input. I focused on creating a responsive design to ensure the app works seamlessly on both mobile and desktop devices. The application also utilizes JavaScript to handle the API calls, ensuring that the data displayed is always current and accurate.

This project sharpened my skills in API integration, JavaScript, and working with asynchronous data. It provided me with a great opportunity to build a practical, real-world web application that offers an engaging user experience while learning how to interact with third-party services effectively.
